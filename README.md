# Kairos Flux | Premium Engineered Scale

A high-end, professional, and minimalist website for **Kairos Flux**, a premium B2B software engineering agency. This project is engineered for absolute scale, focusing on high-performance web architecture and a sophisticated, enterprise-grade aesthetic.

## 🚀 Tech Stack

- **Frontend:** Vanilla JavaScript (ESM)
- **Styling:** Custom Vanilla CSS (Apple-inspired "Bento-box" layout)
- **Animations:** [GSAP](https://gsap.com/) (GreenSock Animation Platform)
- **Smooth Scrolling:** [Lenis](https://github.com/darkroomengineering/lenis)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Typography:** Playfair Display (Serif) & Inter (Sans-serif)

## ✨ Key Features

- **Enterprise Aesthetics:** A dark-themed, minimalist "Bento-box" layout designed for B2B credibility.
- **Advanced Animations:** Subtle micro-interactions, magnetic elements, and scroll-triggered reveals using GSAP.
- **Ultra-Smooth UX:** Integration of Lenis for premium kinetic scrolling.
- **Custom Cursor:** Magnetic cursor dot and ring system for enhanced interactivity.
- **Film Grain Overlay:** A subtle noise texture for a premium, cinematic feel.
- **Responsive Architecture:** Fully optimized for all device scales.

## 🛠️ Development

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shalom-jed/KairosFlux.git
   cd KairosFlux
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `index.html`: Main entry point and semantic structure.
- `style.css`: Custom design system and layout.
- `main.js`: Animation logic, smooth scroll initialization, and interactive components.
- `logo.png`: Visual identity asset.
